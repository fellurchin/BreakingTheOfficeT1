README - RAVVIT.CO - BREAKING THE OFFICE 

La base esencial del juego es destruir todos los objetos posibles, como tal se controla un muñeco con un joystick en pantalla
que se encuentra al lado izquierdo, si este se presiona se activa el movimiento, dependiendo de la direccion
y lo que sea desplazado del centro el joystick el muñeco caminará o correrá, y para realizar la accion de golpear,
se debe presionar un boton blanco que se encuentra en el lado derecho de la pantalla.

Observaciones

En el beta, varias cosas fueron mejoradas
-La primer pantalla que antes decia Start se cambió por una imagen que dice el titulo del videojuego "BREAKING THE OFFICE"
-Se le agregó a esta misma pantalla una seccion de configuracion, que incluye el cambio de graficos, el reseteo del juego,
 y un slider que controlará en actualizaciones posteriores el volumen del audio del videojuego.

-una vez es presionado el titulo del juego se lleva a la seccion de misiones donde se presenta lo siguiente

	-Una mision que cuenta como tutorial del juego, donde no hay tiempo y se le explica al jugador basicamente como moverse
	 y como golpear, se le dan sugerencias y se le reta a desplazarse para que aprenda a controlar el personaje.
	-La segunda mision consta de basicamente destruir una oficina, a este se le da un tiempo para destruir todos los objetos que pueda
	en un tiempo determinado, esto le dará un dinero, y con ello podra comprar mas armas.   

-Cuando el jugador presiona la seleccion una mision, podra elegir entre 6 armas, 3 comprables con el dinero del inicio, y 3 que no podra porque su valor es mayor

En esta version, se consideran cambios que mejoraron
-El desplazamiento del personaje, debido a que el joystick es movil
-La zona de boton de ataque es mayor, asi el usuario no tiene que preocuparse por dar justamente el el boton.
-Se organizaron las diversas pantallas y se mejoro el UI del juego, esto buscando que sea mucho mas comodo para el jugador
-la calidad del juego es guardado, por lo tanto cuando el jugador salga del mismo, no tendra que volver a configurarla.
